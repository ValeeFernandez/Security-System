# 🔐 Sistema de Seguridad Inteligente

Interfaz web desarrollada como parte de un **proyecto de sistema de seguridad basado en Arduino**, diseñado para monitorear condiciones ambientales y detectar presencia de intrusos dentro de un hogar.

Este repositorio contiene únicamente la **interfaz web del sistema**, utilizada para visualizar datos y controlar los módulos electrónicos.

---

## ✨ Descripción del Proyecto

El sistema integra hardware y software para ofrecer monitoreo doméstico en tiempo real mediante sensores conectados a Arduino.

La página web permite:

* Encender y apagar el sistema de seguridad
* Monitorear temperatura y humedad
* Detectar presencia mediante sensor infrarrojo
* Activar o desactivar la alarma
* Visualizar registros históricos de mediciones ambientales

El objetivo fue construir una solución accesible que combinara **IoT + desarrollo web**.

---

## ⚙️ Funcionalidades

✅ Control general del sistema
✅ Lectura de sensor de temperatura y humedad
✅ Detección de movimiento (sensor infrarrojo)
✅ Control de alarma
✅ Registro histórico en tabla dinámica
✅ Interfaz sencilla para monitoreo remoto
